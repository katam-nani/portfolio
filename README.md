# Katam Nani | ECE Portfolio

An animated storyteller portfolio where a cartoon character walks onto a stage, narrates each section of the portfolio, and gestures toward sliding info cards.

## 🌐 Live Site

**[https://katam-nani.github.io/portfolio/](https://katam-nani.github.io/portfolio/)**

> The site is automatically deployed to GitHub Pages whenever changes are pushed to the `main` branch.

## ✨ Features

- 🎭 Animated SVG character that walks on-screen and narrates portfolio content
- 💬 Speech bubble with typewriter effect
- 📋 Sliding info cards for each section (About, Education, Projects, Specialization, Contact)
- ✨ Animated particle background with node connections
- ⏯️ Auto-play story mode with Prev / Pause / Next controls
- ⌨️ Keyboard navigation (`←` / `→` arrow keys, `P` to pause/play)
- 📱 Fully responsive for mobile devices

## 🗂️ Structure

```
portfolio/
├── index.html   # Stage layout and HTML structure
├── style.css    # Dark tech theme + character CSS animations
├── script.js    # Story engine, SVG character, particle background
└── bg.png       # Background image asset
```

## 🚀 Run Locally

```bash
# Any static file server works, for example:
python3 -m http.server 8080
# Then open http://localhost:8080
```
